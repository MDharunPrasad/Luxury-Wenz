Implementation Details

The favicon is implemented using the following code in your HTML <head>:

<!-- Favicon -->
<link rel="icon" href="/favicon.ico" sizes="any">
<link rel="icon" href="/favicon-dark.svg" type="image/svg+xml">
<link rel="apple-touch-icon" href="/favicon-dark.svg">
<link rel="manifest" href="/site.webmanifest">
<meta name="theme-color" content="#0f172a">

<!-- Preload the SVG favicon for better performance -->
<link rel="preload" href="/favicon-dark.svg" as="image" type="image/svg+xml">

<!-- Theme Color for Mobile Browsers -->
<meta name="msapplication-TileColor" content="#0f172a">
<meta name="msapplication-config" content="/browserconfig.xml">

Testing

To verify the favicon is working:

  1. Check your browser tab - you should see the Wenzhomes logo
  2. Add the site to your home screen (on mobile) - it should use the dark theme icon
  3. Check the browser console for any 404 errors related to favicon files
